Авторы игры: ODUHO4KA, Alekto, Dar'ya
Игра находится на стадии разработки. Нынешний её этап можно считать версией 1.0.

//-------------------------------------

Игра для двух игроков. Во время игрового процесса (непосредственно сражение), игроки могут пользоваться только клавиатурой.
Покупка улучшений и создание юнитов осуществляется с использованием "гарячих" клавиш, который можно настроить в настройках игры.

//-------------------------------------

Цель игры: уничтожить вражеский город или заставить противника сдаться (пункт игрового меню "Капитуляция").

//-------------------------------------

Баги не замечены.

//-------------------------------------

Для настройки игры нужно в главном меню выбрать пункт "Настройки". Сохранение новых клавиш происходит сразу после изменения.

//-------------------------------------

Написано на C++, Qt. Для хранения данных используется БД SQLite3 (для хранения статистики игроков) и JSON (для хранения настроек клавиш управления)
